# Fan / IP / Character Risk

Use this reference when the work involves fan art, known characters, animation/game/film IP, logos, mascots, or "in the style of" prompts.

## Style vs Expression

A style is usually broader than copyright protection; a concrete expression is the danger zone.

Lower-risk phrasing:
- "warm hand-painted fantasy adventure style"
- "retro anime-inspired color palette"
- "dark cyberpunk city with neon rain"
- "cute electric creature, original design, no known franchise resemblance"

Higher-risk phrasing:
- Named characters or franchise names.
- Signature costume, props, color blocking, logo, catchphrase, sidekick, or setting.
- "Looks exactly like X but not X."
- A prompt that asks for a legally protected character while changing only small details.

## Practical Viewer Test

Ask: would an ordinary viewer say "this is basically [specific IP/person/character]" within three seconds?

If yes, rate at least Yellow. If the use is commercial, rate Red unless authorization exists.

## Fan Work Risk Pattern

Risk increases as reuse stacks:

1. Name only.
2. Name + personality.
3. Name + personality + relationship network.
4. Visual design / costume / props.
5. Voice / performance / catchphrase.
6. Commercial sale, paid commission, sponsored ad, or merchandise.

The more complete the stack, the harder it is to defend as inspiration.

## Logos and Trade Dress

Treat logos, product shapes, packaging layouts, mascot colors, and brand UI as high-risk in commercial work. Even when copyright is arguable, trademark/unfair competition risk can remain.

## Case Anchors

Use case anchors only as explanatory examples. Verify current facts before publishing exact legal claims, dates, damages, or procedural status.

Useful anchors to verify when needed:
- AIGC character-generation disputes involving famous visual characters.
- Fan-fiction/fan-work disputes involving reused character names, relationships, and commercial publication.
- Major entertainment-company actions against AI image/video tools for generating protected characters.

## Safer Direction

Move from "fan copy" to "inspired original":
- Rename the character and world.
- Change silhouette, proportions, costume, palette, signature props, and powers.
- Replace franchise lore with an original backstory.
- Avoid exact scenes, catchphrases, logos, and side characters.
- Make the concept valuable without the borrowed IP.
