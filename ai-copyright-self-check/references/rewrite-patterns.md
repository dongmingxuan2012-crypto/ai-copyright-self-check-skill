# Rewrite / Remix Patterns

Use this when a work is risky but the user wants to preserve the idea.

## From Known Character To Original Character

Change at least four of:
- Name.
- Silhouette.
- Face shape and hair.
- Costume structure.
- Color palette.
- Signature props.
- Powers/abilities.
- Sidekick/relationship network.
- World rules and backstory.

Better prompt pattern:
> Create an original character inspired by the broad theme of "lonely space courier", avoiding any recognizable franchise costume, logos, names, or signature props.

## From Celebrity Likeness To Fictional Person

Change:
- Face proportions.
- Age, hairstyle, wardrobe, posture.
- Voice and speaking rhythm.
- Catchphrases or performance style.
- Biographical hints.

Add:
> Make the person fictional and not recognizable as any real public figure.

## From Famous Style To Safer Visual Direction

Replace named living artists or protected franchises with descriptive art direction:
- Instead of "Studio X character", use "warm hand-painted family animation, rounded shapes, soft natural light".
- Instead of "famous game UI", use "dark sci-fi tactical interface, grid-based panels, amber status lights".
- Instead of "specific anime hero", use "original shonen-inspired hero with unique costume and emblem-free design".

## From Unlicensed BGM To Safer Audio

Options:
- Replace with commercial-licensed library track.
- Commission or generate original background music.
- Use ambient sound design instead of a recognizable melody.
- Keep the platform-native sound only for that platform if terms allow.
- Remove music from ad versions and keep it only in non-commercial drafts.

## From Fan Work To Commentary

If the user needs to discuss an IP:
- Use screenshots or clips only when necessary and minimal.
- Frame as commentary, critique, news, or education.
- Avoid using the IP as the main selling point for a commercial product.
- Prefer diagrams, original illustrations, or blurred/abstract references.

## Creator-Friendly Response Pattern

Use direct language:

```text
这个点别硬发，风险不在 AI，而在它太像具体 IP。
最快改法：保留"废土少女 + 机械宠物"这个创意，但换掉红蓝配色、圆耳造型、标志道具和角色名。
改完以后，它就从"像某某角色"变成"你的原创角色"。
```
